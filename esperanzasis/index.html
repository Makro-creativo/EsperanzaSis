EsperanzaSis, sistema para realizar pedidos

EsperanzaSis, es un sistema donde podrás ralizar de manera fácil y simple tus pedidos. Contamos con gran variedad de productos y te los llevamos hasta tu casa.

Iniciar sesión
Entrega a domicilio

Con el sistema esperanzaSis, puedes realizar tus pedidos de manera rápida y sencilla y tu pedido llegara hasta las puertas de tu negocio o casa.

Repartidores

Nuestros repartidores haran la entrega del pedido y podrán chequiar el pedido a la hora y fecha en que se entrego

Clientes

Como cliente tendrás acceso al sistema para poder realizar tus pedidos de forma sencilla y rápida.

Panel de control del administrador

Desde el panel de control, el administrador del sitio podrá tener acceso a todos los módulos que tiene el sistema. El sistema cuenta con módulo de pedidos, facturas, proveedores, clientes, etc.

Panel de control del cliente

Desde el panel de control del cliente, el cliente puede realizar sus pedidos. Estos pedidos serán entregados a domicilio por un repartidor.

Panel de control del repartidor

En el panel de control de repartidor, el repartidor podrá ver todos los pedidos realizados al día. También podrá chequiar los pedidos a la hora de ser entregados.
